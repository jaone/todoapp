# todoapp

Todo App with React &amp; Context API

## Installation

```bash
npm i
```

## Usage

```bash
npm run start 
```
or
```bash
react-app-rewired start
```

## Build

```bash
npm run build 
```
or
```bash
react-app-rewired build
```
## Deploy URL

[todoapp-jaone.vercel.app](https://todoapp-jaone.vercel.app/)
